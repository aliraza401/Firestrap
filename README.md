## Quick Start

##### Using CDN

1. Just copy stylesheet link into HTML <head> tag `<link rel="stylesheet" href="https://rawcdn.githack.com/aliraza401/Firestrap/e4bd67f005abbfc2c6a56e57430287289d5a0995/firestrap.css" />`.
2. Start using Fluid
